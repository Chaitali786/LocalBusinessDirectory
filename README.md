
# TheLocalRoot
**Your Neighborhood's Digital Marketplace**

TheLocalRoot is a hyper-local business directory designed to connect residents of **Hagsätra** and **Bandhagen** with home-based entrepreneurs. From artisanal bakers to personal tutors, we bring the local economy to your fingertips.

## Features
* **Area Filtering:** Toggle between Hagsätra and Bandhagen listings instantly.
* **Business Cards:** Clean, professional grid layout showcasing local talent.
* **Responsive Design:** Fully optimized for desktop and mobile (including iPhone 12 mini).
* **Verified Listings:** Highlighting trusted neighborhood partners.
